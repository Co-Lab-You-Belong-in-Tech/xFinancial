# xFinancial - react
> Step-by-step navigation for your financial goals
(aka “GPS for your money”)

## Built with

- React Js
- SASS
- Tailwind


## Getting Started

### Setup

```cmd
 git clone https://github.com/Co-Lab-You-Belong-in-Tech/xFinancial.git
 cd xFinancial
```

### Install

```cmd
npm install
npm run build
```

### Usage

```cmd
npm start
```
